# Devovia Studio Website

Official website for Devovia Studio.

## Stack

- React
- Vite
- CSS
- Netlify

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## Netlify settings

- Build command: `npm run build`
- Publish directory: `dist`
