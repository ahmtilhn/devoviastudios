2. About the App

Arrow Escape is a mobile puzzle game where players follow arrow directions, manage time, use memory, and complete challenging levels.

The app does not include user login, online accounts, membership registration, or cloud profiles.

Game progress and some app settings are stored locally on the user’s device. These local records are not sent to Devovia servers.

3. No User Account or Login System

Arrow Escape does not allow or require users to create an account.

The app does not collect or request:

  • Name or surname
  • Email address
  • Phone number
  • Username
  • Password
  • Profile photo
  • Social media account
  • Personal profile information
  • Identity information

Users can play the game without registering, logging in, or providing personal account information.

4. Locally Stored Data

Arrow Escape may store certain gameplay data locally on the user’s device in order to provide a smooth game experience.

This local data may include:

  • Current level
  • Completed levels
  • Coins
  • Stars
  • Booster counts
  • Daily mission progress
  • Streak reward progress
  • Game settings
  • Sound and music preferences
  • Local gameplay progress

This information is stored only on the device.

Devovia does not upload this local gameplay data to its own servers. The data is not linked to an online user account because the app does not have an account system.

If the user deletes the app, clears app data, changes device, or resets local progress, this local data may be lost.
