1. Introduction

This Privacy Policy explains how Daily Hadith (“the App”, “we”, “us”, or “our”) collects, uses, processes, and protects information when you use our mobile application.

By accessing or using the App, you acknowledge that you have read, understood, and agree to this Privacy Policy.

If you do not agree, you must discontinue use of the App.

2. Scope and Data Controller

For the purposes of applicable data protection laws (including GDPR where applicable), Ahmet İlhan acts as the data controller only for limited data explicitly described in this policy.

We do not act as a data controller for data independently collected by third-party services (e.g., advertising providers).

3. Information We Collect

3.1 Non-Personal Data

We may automatically collect non-identifiable data, including:

  • Device type and model
  • Operating system version
  • App version
  • Usage statistics and session duration
  • Crash logs and diagnostics
  • Language and region settings
  • Time zone

This data is used strictly for performance optimization and stability improvements.

3.2 Location Data (On-Device Processing Only)

The App may request access to location for:

  • Prayer time calculation
  • Qibla direction

📌 Legal clarification:

  • Location data is processed exclusively on the user’s device
  • It is never transmitted, stored, or shared externally
  • We do not track users in real time

3.3 Advertising Data (Third-Party Controlled)

The App integrates:

  • Google AdMob
  • Unity Ads

These providers may collect:

  • Advertising identifiers (GAID / IDFA)
  • IP address
  • Device metadata
  • Ad interaction data

📌 Important legal note:
We do not access, control, or store this data.
All advertising data processing is governed by the respective third-party privacy policies.

3.4 In-App Purchases

All payments are processed by:

  • Google Play Store
  • Apple App Store

We:

  • Do NOT collect payment data
  • Do NOT store financial information

We only receive a transaction token/receipt for verification purposes.

3.5 User Preferences (Local Storage Only)

Stored strictly on-device:

  • Language selection
  • Theme settings
  • Bookmarks
  • Notification preferences
  • Premium status cache

This data is never transmitted externally.

4. Legal Basis for Processing (GDPR Compliance)

Where applicable, we rely on:

  • Legitimate Interest → App functionality & performance
  • User Consent → Permissions (location, notifications)
  • Contractual Necessity → Premium feature activation

5. How We Use Data

We use data solely to:

  • Provide core app functionality
  • Calculate prayer times and Qibla direction
  • Enable notifications
  • Verify purchases
  • Improve app performance
  • Display ads (free version only)

We explicitly DO NOT:

  • Sell personal data
  • Build user profiles
  • Track users across apps
  • Use data for unrelated purposes
